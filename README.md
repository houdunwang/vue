## 安装项目

**命令安装**

首先全局安装 houdunren-vue

```
npm install -g houdunren-vue
```

然后执行命令进行安装 houdunren-vue

```

```

**Git 下载**

使用 git 下载项目也是可以的

```
#github
git clone https://github.com/houdunwang/vue.git

#gitee
git clone https://gitee.com/houdunren/vue.git
```

安装依赖包

```
#npm
npm install

#yarn
yarn

#pnpm
pnpm install
```
